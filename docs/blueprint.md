# **App Name**: Sahayak PWA

## Core Features:

- Authentication & Authorization: User authentication and role-based access control using Firebase Auth (teachers and students).
- Role-Based Dashboards: Role-specific dashboards (teacher and student) with quick access, recent activities, and announcements.
- AI Content Generation: AI-powered tool to generate hyper-local content. Includes voice input, output editing, and language selection.
- Visual Aid Designer: AI-powered tool to generate visual aids from text descriptions. Includes style selection and variation generation.
- AI Question Answering: Instant knowledge base where AI generates explanations and analogies for questions, supports voice input.
- Content Management: Content organization (Stories & Explanations, Generated Worksheets, Saved Visual Aids)
- Settings & Support: User profile editing, notification settings, help and support resources, legal and privacy information.

## Style Guidelines:

- Primary color: HSL(48, 90%, 50%) - RGB(#F0B90B), A vibrant yellow that symbolizes knowledge and energy. Used for primary interactive elements such as the generate buttons or active tab indicators. This is a color suitable for a younger audience and intended to be inviting and supportive
- Background color: HSL(48, 20%, 90%) - RGB(#F5F2E8). A desaturated, light yellow which helps reduce eye strain on a display.
- Accent color: HSL(18, 70%, 40%) - RGB(#B34D0F). A dark reddish-brown used sparingly for contrast and to draw attention to important actions. Examples of where the accent color is applied would be errors, confirmations, and/or warnings.
- Headline font: 'Poppins', a geometric sans-serif with a precise, contemporary, fashionable, avant-garde look
- Body font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality.
- Simple, flat icons that are easy to understand, with an outline style, ensuring they are easily recognizable even in low-bandwidth conditions.
- Mobile-first, responsive design using CSS Grid and Flexbox. Focus on minimal image sizes and lazy loading.
